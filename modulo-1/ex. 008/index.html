Principais Formatações

Negrito / Destaque:

→ Tag "b" (ñ semântica): Aqui, temos um termo destacado com a tag (b). A diferença é que essa tag visa comente a forma, e não a semântica.
→ Tag "strong" (semântica): Aqui, temos um termo em destaque, destacado pela tag (strong). Sua diferença é que essa tag visa a semântica, ou seja, o significado do seu uso.

Itálico / Ênfase:

→ Tag "i" (ñ semântica): Exemplo de termo em itálico com a tag (i), e usando o comando de envelope (Ctrl+Shift+P).
→ Tag "em" (semântica): Exemplo de termo enfatizado com a tag (em).

Marcador:

Para marcar o texto como se fosse um marcador stabilo, basta utilizar a tag "mark" - que só pode ser utilizada envelopando o texto ou abrindo a tag.
Ex.: Lorem ipsum dolor, sit amet consectetur adipisicing elit. Ut natus, facere, eum omnis non vitae magnam impedit, ea culpa cupiditate exercitationem! Facere necessitatibus ab architecto, adipisci nemo accusamus soluta numquam.

Cor:

Agora, para mudar a cor da marcação, podemos usar um parâmetro "style" ou uma tag "style" no head.
→ Parâmetro: Exemplo de texto utilizando o parâmetro, irei usar o roxo
→ Head: Exemplo de texto usando o head, marcação a partir daqui.
